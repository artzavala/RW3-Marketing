
    2. Install production dependencies:
    ```bash
    npm install @clerk/nextjs drizzle-orm @neondatabase/serverless svix
    ```

    3. Install dev dependencies:
    ```bash
    npm install -D drizzle-kit tsx dotenv
    ```

    4. Initialize shadcn/ui:
    ```bash
    npx shadcn@latest init -d
    ```
    The `-d` flag uses defaults (New York style, neutral color, CSS variables).

    5. Add shadcn/ui components needed for the navigation shell:
    ```bash
    npx shadcn@latest add sidebar button avatar dropdown-menu separator sheet tooltip
    ```

    After scaffolding, verify the dev server starts:
    ```bash
    npm run dev &
    sleep 5
    curl -s -o /dev/null -w "%{http_code}" http://localhost:3000
    kill %1
    ```
    Expected: HTTP 200.
  </action>
