2. Create `src/lib/google/sheets-client.ts` with three exported functions:

**extractSpreadsheetId(url: string): string | null**
- Regex: `/\/spreadsheets\/d\/([a-zA-Z0-9-_]+)/`
- Returns match[1] or null

**getSheetsClient()**
- Reads `process.env.GOOGLE_SHEETS_CREDENTIALS_BASE64`
- Throws descriptive error if not set
- Decodes base64: `JSON.parse(Buffer.from(raw, 'base64').toString('utf-8'))`
- Creates `google.auth.GoogleAuth` with credentials and scope `https://www.googleapis.com/auth/spreadsheets.readonly`
- Returns `google.sheets({ version: 'v4', auth })`

**readSheetRows(spreadsheetId: string, tabName: string): Promise<string[][]>**
- Calls `sheets.spreadsheets.values.get({ spreadsheetId, range: \`${tabName}!A:Z\` })`
- Returns `res.data.values ?? []`
- Does NOT skip the header row (caller decides how to use rows)
